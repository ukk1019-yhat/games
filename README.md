## 🌟 What is this?

**Retro DOS Game Vault** is a love letter to the golden era of Chinese PC gaming — the 90s and early 2000s, when floppy disks and dial-up internet ruled. Step into a living museum of classic titles, fully playable in your browser, with zero setup.

No downloads. No installs. Just click and play.

---

## 🎯 Features

| 🚀 Feature | 📝 Description |
|:---|:---|
| **Instant Play** | Launch any game directly in-browser via DOSBox emulation |
| **Smart Search** | Find games by name, genre, year, or developer |
| **Mood Browser** | Explore by vibe — relaxing, nostalgic, challenging, weird |
| **Game of the Day** | A daily spotlight with history & trivia about a featured title |
| **Favorites & Progress** | Save your favorite games and resume where you left off |
| **Mobile Friendly** | Touch controls for playing on phones & tablets |

## 🖥️ Quick Start

### 1️⃣ Clone & Install

```bash
git clone <your-repo-url>
cd retro-dos-game-vault
npm install
```

### 2️⃣ Fetch Game Data

```bash
python download_data.py
```

> ⚠️ Having trouble downloading? Check the **Issues** tab for known fixes.

### 3️⃣ Launch

```bash
npm run dev
```

Open **`http://localhost:3000`** and start exploring 🎮
